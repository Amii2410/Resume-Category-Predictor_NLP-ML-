# Resume Category Predictor (NLP + ML)

This Streamlit web application predicts the job category of a resume using **Natural Language Processing (NLP)** and **Machine Learning**.

Users can upload a resume PDF or paste resume text, and the app will automatically classify it into a job category.

---

## Features
- Upload resume PDF
- OCR support for scanned resumes
- Paste resume text
- Text preprocessing using NLP
- ML model using TF-IDF + SVM
- Simple and clean UI
- Deployed on Streamlit Cloud

---

## Deployment
This app is deployed using **Streamlit Cloud** and can be accessed online.
