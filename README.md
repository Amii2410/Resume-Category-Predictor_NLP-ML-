# Resume Category Predictor (NLP + ML)

This Streamlit app predicts the job category of a resume.

## Features
- Upload resume PDF
- OCR support for scanned resumes
- ML model using TF-IDF + SVM

## Deployment
This app is deployable on Streamlit Cloud.

## Files
- app.py → Streamlit app
- resume_classifier_model.pkl → Trained ML model
- requirements.txt → Python dependencies
- packages.txt → OCR system dependencies
